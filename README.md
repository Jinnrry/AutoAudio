# AutoAudio
> 安卓音量管理工具

### 下载

[APK程序下载](https://github.com/jiangwei1995910/AutoAudio/releases)

### 初衷

上班狗、大学狗专用音量管理工具，设置好上班上课时间，上班上课期间自动改变手机音量，下班下课自动恢复音量。为你解决上课开会突然手机响，下班下课错过铃声的烦恼。
